## Description

[Node-Nest](https://github.com/dbacilio88/node-nest) Application TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Dev docker

```bash
# image docker build
$ docker compose build  

# docker run container
$ docker compose up
```

## Prod docker

```bash
# image docker build
$  docker compose -f docker-compose.prod.yml build   

# docker run container
$  docker compose -f docker-compose.prod.yml up
```
